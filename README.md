# Voice Assistant using python
## Description
This project is a simple voice assistant built using python
It listen to voice command and respond to the command by speech
## Features
-It greet the user
-Open application(Calculator,Notepad)
-Open websites(youtube,Google)
-Tells current date and time
-Voice based interaction
## Technologies Used
-Python
-SpeechRecoginition
-pyttsx3
## How to run
-Install python
-Intall libraries
    pip install speechrecoginition pyttsx3 pyaudio
## Run the program
py assistant.py
